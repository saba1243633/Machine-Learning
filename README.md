# Machine-Learning

# Objective
This dataset aims to analyze factors influencing student placements. It includes academic performance, extracurricular activities, skills, and placement training to determine their impact on placement status.

# Overview
The dataset of 10,000 students analyzes factors affecting placements, including academics (CGPA, SSC, HSC marks), skills (internships, projects, certifications, aptitude, soft skills), extracurriculars, and placement training.
